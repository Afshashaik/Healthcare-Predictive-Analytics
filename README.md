# Healthcare Predictive Analytics (Disease Detection)

## Overview

This project predicts diabetes risk using Machine Learning algorithms and healthcare analytics. The model analyzes patient medical data such as glucose level, BMI, insulin, blood pressure, and age to determine whether a patient is likely to have diabetes.

The project demonstrates data preprocessing, normalization, exploratory data analysis (EDA), classification modeling, feature importance analysis, and healthcare-focused predictive analytics.

## Objective

To build a Machine Learning model that predicts diabetes risk using patient medical records and classification algorithms.

## Dataset

Dataset Used:

* Diabetes Dataset from Kaggle

Dataset Features:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
* Outcome

The models were evaluated using:

* Accuracy Score

## Feature Importance Analysis

Feature importance analysis was performed using Random Forest to identify the most influential medical factors affecting diabetes prediction.

Important features included:

* Glucose
* BMI
* Age
* Insulin

## Ethical Data Handling & Patient Privacy

This project uses anonymized healthcare data for educational and research purposes only. Patient privacy and ethical data handling practices were considered throughout the analysis. No personally identifiable information (PII) was used.
